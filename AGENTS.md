# Project: raycast-ollama

This project provides Raycast extensions for Ollama.

## Project Structure

* `src/*.ts` - Contains all TypeScript files for extension commands.
* `src/lib/mcp` - Contains the Model Context Protocol (MCP) class used by commands and UI files.
* `src/lib/ollama` - Contains the Ollama class used by commands and UI files.
* `src/lib/settings` - Contains settings functions used by commands and UI files.
* `src/lib/ui` - Contains UI functions used by command files.

## Coding Standards

* Use TypeScript.
* Use  `npm run fix-lint` command to comply with linting rules and automaticaly fix.
